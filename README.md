## How to get started

1.  To make a move, the player will use the mouse single click to mark a space. In this version, there is no provision to undo the move. As soon as the move is done, the game proceeds to the next playerâ€™s chance.
1.  At each move, The game will show whose move (currentPlayer X or currentPlayer 0) it is. When the game ends, it displays Player won or Game Draw .
  - Winner: currentPlayer X
  - Winner: currentPlayer 0
  - Draw
